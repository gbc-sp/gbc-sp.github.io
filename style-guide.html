Combination Headings

When an h1 or h2 immediately follows an h3, their top margin colapses to bring the elements closer together vertically.

<h3>Subheading</h3>
<h1>Primary Heading</h1>

Text Elements

Lists

Ordered & Unordered

Lists created in the following ways: 
<ol>
  <li>First</li>
  <li>Second</li>
  <li>Third</li>
</ol>
<ul>
  <li>Item</li>
  <li>Item</li>
  <li>Item</li>
</ul>

Result in the following lists:

  1. First
  2. Second
  3. Third
  • Item
  • Item
  • Item

Ordered & Unordered

Style can be added back using .counted and .bulleted for numbers and bullet points respectively.

<ol class="counted">
  <li>First</li>
  <li>Second</li>
  <li>Third</li>
</ol>
<ul class="bulleted">
  <li>Item</li>
  <li>Item</li>
  <li>Item</li>
</ul>

With the following results

  1. First
  2. Second
  3. Third
  • Item
  • Item
  • Item

Inline lists

Lists can be set inline by applying .list-inline to the list

<ul class="bulleted">
  <li>Item</li>
  <li>Item</li>
  <li>Item</li>
</ul>

This results in the following:

  • Item
  • Item
  • Item
